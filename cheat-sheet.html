JavaScript Objects Cheat Sheet

Object Initializer 
var genericObject = {
    prop1: value1,
    prop2: value2,
    "prop n": val_n };
 We can create new objects using object initializer aka using literal notation. In the same genericObject is the name of the new object and each property is listed followed by a colon and its assigned property value. This is similar to calling new Object() and the properties assigned can be added to or removed from later down the line.
Constructor Function 
function SomeObject(val1, val2, val3) = {
    this.prop1 = val1;
    this.prop2 = val2;
    this.prop3 = val3;
    }

var sampleObj = new SomeObject("value", "val", "v");
With constructor functions we create a function that enables us to call it passing in arguments which represent the property values of our object. In doing so, we are able to create several instances of the same type of object using this constructor as a template.
